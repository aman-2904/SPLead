# ShaadiPlatform Feature Directories

This folder contains scale-ready directories representing individual business modules:

- `/auth` - Handles bride, groom, and vendor authentications.
- `/vendors` - Vendor directory search, filtering, detail views, and reviews.
- `/checklist` - Planning tasks management and coordination.
- `/budget` - Cost allocations, track spent funds, and custom calculations.
- `/guestlist` - Guest RSVPs, table seat assignments, and contact records.

Each folder should hold its own subcomponents, page routing subfolders, or specific feature hooks.
